- 팀명 : Persona
- 프로젝트명 : Persona Frame

## <배경 및 목적(Background & Goals)
### 문제 제기
- 웹 기반 AI의 한계
    - 챗봇, 웹 서비스 형태의 AI는 대화와 정보 처리 능력은 뛰어나지만, 항상 브라우저 접속을 전제로 함
    - 사용자는 필요할 때마다 탭을 열고 질문한 뒤, 답변을 확인하면 곧바로 세션이 종료되는 단발성 사용 패턴에 머물 수 있음
- 스마트 스피커의 한계
    - 스마트 스피커는 음성 명령에는 적합하지만, 화면이 없거나 제한적이어서 시각 정보를 충분히 표현하기 어려움
    - 사용자 패턴 분석이나 장기적인 맥락을 반영한 상호작용에는 제약이 큼

### 기대 효과
- 통합 인터페이스 제공
    - 음성, 시각 정보, 알림을 하나의 전자액자 화면에서 처리함으로써, 여러 기기 / 앱을 오가던 작업을 단일 디바이스로 묶을 수 있음
- 지속적인 사용 경험 형성
    - 필요할 때만 호출하는 AI가 아니라 일상 속에서 수시로 마주치는 상주형 인터페이스를 구현할 수 있음
- 로컬 패턴 분석 기반 개인화
    - AI 로컬 서버에서 사용자의 일정, 사용 시간대 등을 분석하여, 웹 서비스 수준을 넘어서는 사용자 맞춤형 안내 / 회고 / 제안이 가능해짐
- 음성 / 시각 정보 결합에 따른 이해도 향상
    - 스마트 스피커의 음성 출력에 웹 화면 수준의 시각 정보를 결합함으로써 복잡한 일정/요약 내용을 더 직관적으로 이해할 수 있음
- 프라이버시 관점의 대안 제시
    - 개인 일정 등의 데이터를 로컬 디바이스에서 처리함으로써 모든 정보를 클라우드에 저장하는 구조 대비 프라이버시 친화적인 AI 사용 방식을 검증할 수 있음

## <대상 사용자(Target Audience)>
- 전 연령대를 아우르는 범용적 접근성
- 정서적 교감과 유대감을 지향하는 1인 가구
- 디지털 기기 조작에 익숙하지 않은 디지털 취약계층 및 고령층
- 텍스트 입력의 번거로움 없이 구두(口頭)를 통한 직관적 일정 관리를 선호하는 사용자

## <ngrok 임시 배포>
> [!info]
> ### ngrok 설치
> MAC: `brew install ngrok/ngrok/ngrok`
> Windows: `choco install ngrok`
> 
> ### 토큰 등록
> `ngrok config add-authtoken 여기에_발급받은_토큰_붙여넣기`
> 
> ### HTTP 서버 실행
> `cd 경로`
> `python3 -m http.server 원하는_포트번호`
> 
> ### 새 터미널 창 열고 ngrok 실행
> `ngrok http 포트번호`
> 
> ### 브라우저에서 접속
> -  https://xxxx-xxxx.ngrok-free.app/페이지명
> - 웹 서버는 기본적으로 폴더 접속 시 index.html을 자동으로 엶(다른 파일 열고 싶으면 뒤에 `/파일명` 적기) ex) https://xxxx-xxxx.ngrok-free.app/mobile.html
> 
> ### 종료
> 터미널 끄기

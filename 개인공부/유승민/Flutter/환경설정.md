## <iOS 개발>
### Xcode 설치
- 설치 후 라이선스 동의 : `sudo xcodebuild -license accept`
- Command Line Tools 설정 : `sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer`


### CocoaPods 설치
- `brew install cocoapods`

### iOS 시뮬레이터 설치
- Xcode 열고 Settings → Platforms → iOS에서 원하는 버전 다운
아니면
- `xcodebuild -downloadPlatform iOS`

## <`.config/flutter` 폴더 권한 문제>
- brew로 설치하면 가끔 `.config` 폴더를 Flutter가 못 만드는 경우도 있음
### 폴더 만들고 권한 부여
- `sudo mkdir -p /Users/yooseungmin/.config/flutter`
- `sudo chown -R yooseungmin /Users/yooseungmin/.config`

### 다시 실행
- `flutter doctor`

## <프로젝트 생성 및 실행>

```
open -a Simulator
flutter run
```
